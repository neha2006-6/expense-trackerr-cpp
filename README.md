#  Expense Tracker in C++

A simple and useful C++ application to track 
your daily expenses!

##  Features
- Add expenses with category and description
- View all expenses with total
- Filter expenses by category
- Delete expenses
- Calculate total spending

##  How to Run

### Step 1 - Compile
```bash
g++ expense_tracker.cpp -o expense_tracker
```

### Step 2 - Run
```bash
./expense_tracker
```

##  Technologies Used
- C++
- Vectors
- Structs
- Standard Template Library (STL)

##  Requirements
- C++ compiler (g++)
- Any operating system (Windows, Linux, Mac)
